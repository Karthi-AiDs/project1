Project: Construction Management System

Description: The Construction Management System app, designed in Figma, is a clean and user-friendly interface created to simplify the management of construction projects. It features key screens such as a dashboard for project overviews, task manager for assigning and tracking work, team view for managing member roles, and reports for monitoring budgets and progress. The design also includes a smooth login/register flow with role selection. Built with a modern layout and responsive design, the app aims to provide an efficient experience for project managers, engineers, and clients.
